Manual do Usuário: Sistema de Cálculo de Pontuação de Rummikub

1. Introdução

Este sistema foi desenvolvido para auxiliar na contagem de pontos durante as partidas de Rummikub, automatizando o processo de cálculo e garantindo que as regras sejam seguidas corretamente. O sistema permite a inserção das pontuações de cada jogador em cada rodada e realiza o cálculo necessário, destacando os maiores valores e sorteando o jogador da mão.

2. Interface do Sistema

  • Tabela de Pontuação: A tabela central da interface permite a inserção das pontuações de cada jogador em cada rodada. Cada linha representa uma rodada e cada coluna, um jogador.
  • Botão Calculadora: Após inserir as pontuações, pressione o botão de calculadora para que o sistema processe as pontuações de acordo com as regras estabelecidas.

3. Funcionalidades Principais

3.1 Inserção de Pontuações

Para cada jogador, insira a pontuação obtida na respectiva rodada. Se o jogador não pontuar, deixe o campo em branco ou insira "0".

Ao inserir um valor, o sistema considerará o número como positivo, representando o ganho de pontos.

3.2 Cálculo Automático da Rodada

O cálculo da rodada será realizado apenas quando houver um único jogador sem pontuação ou com a pontuação "0".

Se houver mais de um jogador sem pontuação, o sistema emitirá um alerta indicando que é necessário completar as pontuações antes de proceder.

Ao calcular, o sistema:

  • Converterá todas as pontuações dos jogadores que preencheram valores em números negativos.
  • Somará esses valores e atribuirá a soma ao jogador que estava sem pontuação.

3.3 Verificação de Recalculo

O sistema possui uma verificação que impede o recálculo de uma rodada que já foi processada, evitando alterações indesejadas.

Caso uma rodada já tenha sido calculada e o usuário tente recalcular, será exibida uma mensagem informando que a rodada já foi calculada.

3.4 Sorteio do Jogador da Mão

Após cada rodada, o sistema automaticamente define o próximo jogador da mão, seguindo a ordem de rotação definida.

Caso as pontuações sejam apagadas, um novo sorteio será realizado para definir o jogador da mão.

4. Alertas e Mensagens

  • "Essa rodada já foi calculada.": Aparece quando se tenta recalcular uma rodada já processada.
  • "Não é possível calcular a rodada. Ainda existem jogadores sem pontuação.": Aparece quando há mais de um jogador sem pontuação na rodada.

5. Dicas de Uso

  • Complete todas as pontuações: Sempre verifique se todos os jogadores, exceto um, têm suas pontuações preenchidas antes de calcular.
  • Revisão após o cálculo: Após o cálculo de uma rodada, revise as pontuações para garantir que os valores negativos e positivos estejam corretos.

6. Solução de Problemas

  • Problema: "O sistema não calcula a rodada."
    • Solução: Verifique se há mais de um jogador sem pontuação. O sistema não permitirá o cálculo se esse for o caso.
  • Problema: "Recebo uma mensagem dizendo que a rodada já foi calculada."
    • Solução: Isso ocorre quando você tenta recalcular uma rodada que já foi processada. Se precisar fazer ajustes, certifique-se de que todas as pontuações sejam apagadas antes de tentar novamente.

7. Conclusão

Este sistema foi criado para facilitar a contagem de pontos no Rummikub, permitindo que os jogadores se concentrem no jogo sem se preocupar com cálculos complexos. Siga as instruções deste manual para garantir uma experiência tranquila e eficiente.
